
# 安装

1. 下载仓库

```
git clone https://github.com/Yukiriri/win-redist-fix.git
```

2. 进入dotnet和vc文件夹后运行对应版本的bat会下载对应的运行库安装包（你可以自己选择要下载哪些版本）

3. 管理员运行"install.bat"

# 更新

```
cd win-redist-fix
git pull
```
